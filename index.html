贴吧云签到

实现不登录电脑的全自动云签到

Download this project as a .zip file Download this project as a tar.gz file

这是一款节省数据库又具有高效率的百度贴吧云签到程序
云签到，即不用登录电脑，也能自动签到，它就是这么方便。
它，比同类软件更快，更轻，更省数据库；特别设计的响应式界面让您在任何终端上都能愉悦地使用它。

这是一款开源程序，你可以在遵守GPLv3协议的前提下任意修改和发布它。
